RagWeb文件夹
【网页用】
Flask框架下网页VS2017工程

代码在子目录\RagWeb\RagWeb下：
------------------------------------------------------------------------------------
网页内容文件
RagWeb文件夹：
	static：存放图片、音乐、css、js等静态内容
	templates：存放html文件（input.html/output.html）
	myform.py：表单类
	views.py：关于网页的路由函数

------------------------------------------------------------------------------------
其他文件：
env：虚拟环境的文件夹
model：诗文配对情感模型文件夹
word_vec_model：词向量模型
runserver.py：本地网页运行文件
requirements.txt：环境依赖的python包的各种要求（等等等等忘记整理了）
